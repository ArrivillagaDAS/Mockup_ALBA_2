# Mockup ALBA 🎬

¡Bienvenido al repositorio **Mockup_ALBA_2**! Este proyecto es una réplica visual (mockup) enfocada en el diseño de interfaces limpias, modernas y totalmente responsivas, aplicando las mejores prácticas de maquetación web.

---

## 🚀 Características del Proyecto

*   **Diseño Responsivo:** Adaptabilidad total a dispositivos móviles, tablets y pantallas de escritorio.
*   **Interfaz Moderna:** Uso de layouts avanzados (Flexbox / CSS Grid) para una estructura sólida.
*   **Componentes Interactivos:** Menús interactivos (como menús hamburguesa para móviles) y transiciones fluidas.
*   **Código Limpio:** Estructura de archivos organizada y semántica HTML5 estricta.

---

## 🛠️ Tecnologías Utilizadas

*   **HTML5:** Estructuración semántica del contenido.
*   **CSS3:** Estilos, animaciones personalizadas, variables de entorno y Media Queries para el diseño *responsive*.
*   **JavaScript (Opcional/Básico):** Manipulación del DOM para interacciones del usuario (como abrir/cerrar menús o modales).

---

## 📁 Estructura del Repositorio

El proyecto mantiene una estructura organizada para facilitar su mantenimiento:

```text
Mockup_ALBA_2/
├── index.html          # Página principal del proyecto
├── css/                # Hojas de estilo
│   └── styles.css      # Estilos globales y layouts
├── js/                 # Lógica e interactividad
│   └── main.js         # Scripts del DOM
└── assets/             # Recursos estáticos (Imágenes, iconos, etc.)
```

## 📌 Buenas Prácticas Aplicadas

*  Control de Versiones: Uso estratégico de Git para el flujo de trabajo (branching y commits claros).
*  Semántica: Marcado HTML que mejora la accesibilidad y el SEO básico.
*  Modularidad: Separación limpia de responsabilidades (Estructura, Estilos y Lógica).
